# Email-messenger

Chat using e-mails!!!

## Overview

Email-messenger is a Java-based messaging application that enables real-time chat functionality using email as the underlying communication protocol. This project aims to provide a seamless chat experience while leveraging the reliability and ubiquity of email infrastructure.
